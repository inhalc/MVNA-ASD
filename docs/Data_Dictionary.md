# Data Dictionary

## events.tsv (BIDS)
| Column | Description |
| --- | --- |
| onset | Event onset in seconds |
| duration | Event duration in seconds |
| trial_type | Condition label |
| accuracy | Behavioral accuracy (0/1) |
| response_time | Response time in seconds |
| stimulus_id | Stimulus identifier |
| run | Run index |
| block | Block index |
| trigger | Original trigger code |

## participants.tsv (BIDS)
| Column | Description |
| --- | --- |
| participant_id | Subject ID (e.g., sub-001) |
| age | Age in years |
| iq | IQ score |
| diagnosis | Diagnosis label |
